💱 全链路货币结构与三段风险
基准币
CNY
我方记账与最终收益结算的本位货币。所有利润最终折算回基准币衡量。
供应商报价币
CNH / USD / …
供应商开票并收款的货币，可多选。备货付款时需将基准币兑换为供应商报价币。
销售结算币
MXN
墨西哥客户支付的货币，速卖通平台以 MXN 结算后换汇回基准币。
全链路三段风险：
采购换汇风险
基准币 → 供应商报价币
备货付款时，用基准币换供应商报价币产生的汇率风险。选用 CNY/CNH 报价可消除此段风险。
→
成本定价隐含风险
供应商报价币 → 销售结算币（MXN）
采购成本以供应商报价币锁定，但售价以 MXN 定价，两者汇率变动影响实际利润率。
→
收款换汇风险
销售结算币（MXN）→ 基准币
MXN 到账后换回基准币，帐期内 MXN 汇率波动直接影响最终收益，是最主要的敞口。
📦 业务模式
公司提前预测市场需求，选品采购后发往墨西哥自有仓库，在速卖通等本地电商平台售卖并对客交收，以销售结算币（MXN）收款后换汇回基准币（CNY）。
⚠️ 货在墨西哥仓库期间，预期收款以销售结算币（MXN）结算，库存周转速度直接决定 收款换汇风险（MXN → 基准币）的暴露窗口长度。
📐 当前粗估方式
帐期 = 运输在途 + 清关入仓 + 平台出售 + 平台结算
「平台出售」节点当前用行业经验值（海运14天/空运10天）代替实际库存周转数据，为保守估算。
🗓 待升级：当有 SKU 级库存周转数据后，将「平台出售」天数替换为实际平均库存周转天数，汇率风险估算精度将大幅提升。
